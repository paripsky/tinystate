Installation

You can install tinystate using npm or yarn:

npm install tinystate-react
# or
yarn add tinystate-react

Usage

To create a store, use the createStore function from the tinystate library. You'll need to provide an initial state for your store:

// Import necessary functions
import { createStore, createUseStore } from 'tinystate';

// Create a store
const countStore = createStore({ initialState: 0 });

// Create a custom hook to use the store
export const useCountStore = createUseStore(countStore);

After creating a store, you can use the useCountStore hook in your components to access and update the state.

API Reference

The following methods are available for managing the state in your store:

// Subscribe a listener to state changes
subscribe(fn: Listener): () => void;

// Get the current snapshot of the state
getSnapshot(): T;

// Update the state and notify subscribers
setState(newState: T): void;
